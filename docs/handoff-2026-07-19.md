# voice_bbs 引き継ぎメモ (2026-07-19)

## 基本情報
- **URL**: https://bubblevoice-941146906074.asia-northeast1.run.app
- **GitHub**: https://github.com/bonsai/voice_bbs
- **GCP Project**: yok-ai-2026
- **Cloud Run**: asia-northeast1, 512Mi, 1vCPU, 0-2 instances

## 完了した機能

### 1. ランディングページ (`/`)
- ステップバイステップオンボーディング (logo → explanation → mic check → rooms)
- ルームタイル (ランダム配置+回転)
- +NEWボタン

### 2. ルームページ (`/room/:id`)
- 音声録音・再生
- バブルUI

### 3. しりとり (`/shiritori`)
- ASR入力 + 手入力
- 仮名チェーンバリデーション
- グレースケールエフェクト (バリデーション失敗時)
- ゲームオーバー検出

### 4. 管理パネル (`/manage`)
- posts/status/roomsタブ
- roomsタブ: カラフル四角タイル + open/closedトグル

### 5. API
- POST /api/upload (音声PNG投稿)
- GET /api/tts (TTS)
- GET /api/rooms (ルーム一覧)
- 認証: publicモード

### 6. UI機能
- 接続切断時グレースケールオーバーレイ
- 風船アニメーション (wobble, rainbow shimmer)

## データ構造
- `rooms.json`: ルームメタデータ (id, name, type, open)
- `posts` テーブル: 音声投稿
- `shiritori` テーブル: しりとりチェーン

## デプロイ方法
```bash
cd MEGA/voice_bbs
git add -A && git commit -m "message" && git push origin master
gcloud run deploy bubblevoice --source . --region asia-northeast1 --platform managed --allow-unauthenticated --project yok-ai-2026
```
